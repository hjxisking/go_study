思路：
    格雷编码是有规律的
    n = 1
        => 0
           1

    n = 2
        => 0|0
           0|1
            --
           1|1
           1|0

    n = 3
        => 0|00
           0|01
           0|11
           0|10
            --
           1|10
           1|11
           1|01
           1|00

    n - 1
        => 0|grayCode(n-1)
           1|倒序(grayCode(n-1))  == 倒序对应的grayCode + 2^n
