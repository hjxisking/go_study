思路：
    slow和fast同时出发，slow走1，fast走2，假设进环之前的距离是a，进环之后相遇点离进环点的距离是b，相遇点再走c距离又到进环点
    假设相遇前fast已经走了n圈环了，那么
        fast走过的总长度：a + n(b+c) + b => a + (n+1)b + nc
        slow走过的总长度为：a + b
    由于fast = 2 * slow，得到：a + (n+1)b + nc = 2(a + b)，a = c + (n-1)(b+c)
    这个公式的意思是，从头走到入环点的距离等于走n-1圈环再加上c，而从当前相遇点如果走n-1圈再加c的距离正好也是入环点
    所以，当相遇点继续走，同时头部也多一个指针继续走，在新指针走了a距离后，相遇点走了n-1圈+c的距离后，正好会相遇（因为距离一样），且相遇的点就是入环点
